# 계산기
사칙연산 및 나머지 연산을 문자열로 입력받아 수행하는 계산기입니다.

## 목차
- [기능](#기능)
- [사용법](#사용법)

## 기능
-   **덧셈 (+)**: 두 숫자를 더합니다.
-   **뺄셈 (-)**: 두 숫자를 뺍니다.
-   **곱셈 (\*)**: 두 숫자를 곱합니다.
-   **나눗셈 (/)**: 두 숫자를 나눕니다. (0으로 나누기 예외 처리 포함)
-   **나눈 나머지 (%)**: 두 숫자를 나눈 나머지를 구합니다.
-   **종료(e)** 

## 사용법
- Application을 실행하여 키보드 입력을 통해 사용
- Test 코드를 수행하여 수행 결과를 확인
